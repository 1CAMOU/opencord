# OpenCord

A censor-free, discord-like web app based on the Nostr Protocol.
Currently W.I.P

## Features

- [ ] Login
- [ ] Direct Messages (Encrypted)
- [ ] Group Chats (Channels)
- [ ] Contacts
- [ ] Send Images
- [ ] Link Previews
- [ ] Profile Bio
- [ ] Channel Permissions
- [ ] Search
- [ ] Notifications
